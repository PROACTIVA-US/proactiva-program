# Input Contract — Decision Audit

Invocation must provide:

## Required
- Invocation ID
- Decision question
- Candidate (thing to evaluate for reuse)
- Target (thing you are building)
- Access method (local path or URL)
- Constraints ("must nots", locks, boundaries)

## Optional
- Desired posture bias
- Known risks
- Time budget
