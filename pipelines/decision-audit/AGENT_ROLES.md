# Agent Roles — Decision Audit

## Program Director (human)
- Approves / modifies / rejects the final decision.
- Issues binding directives.

## Project CEO (agent)
- Confirms intent + boundaries.
- Ensures reporting is complete and safe.
- Submits the Decision Report upward.

## Executor (agent)
- Performs the audit.
- Drafts the decision memo.
- Proposes extraction boundaries if needed.

## Reviewer (agent)
- Checks coupling risk, lock violations, unsafe language.
- Approves / conditions / rejects the memo.

## Researcher (optional)
- Supplies domain facts, benchmarks, comparable systems.
- Fact-checks executor claims (cite sources if used).
