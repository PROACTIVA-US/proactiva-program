# Review Command

Review the current state of work and suggest improvements.

## Instructions

When the user runs `/review`, perform these checks:

1. **Structure Check**: Verify all expected directories and files exist
2. **Decision Log**: Check if recent decisions have been captured
3. **Workflow Alignment**: Confirm work follows the canonical flow
4. **Constraint Compliance**: Verify HVAC safety boundaries are respected

## Review Checklist

- [ ] phase-1/ structure is intact
- [ ] Decision log is up to date
- [ ] No compliance claims in documentation
- [ ] HVAC outputs marked as assistive-only
- [ ] Human confirmation requirements documented

## Output Format

Provide:
1. Summary of current state
2. Any issues found
3. Recommended next steps
